How to Play Minesweeper 2.0

1. Choose a difficulty level:
• Easy: 9x9 grid with 10 mines.
• Medium: 16x16 grid with 40 mines.
• Hard: 16x30 grid with 99 mines.

2. Left-click a cell to reveal it. If it contains a mine, you lose.
- If the cell is empty, a number shows the count of adjacent mines.
- If the number is 0, adjacent cells are automatically revealed.

3. Right-click (o “long press” su dispositivi touch) per contrassegnare una cella con un flag se sospetti una mina.

4. Vinci scoprendo tutte le celle prive di mine!
